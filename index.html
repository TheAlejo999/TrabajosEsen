Descripcion del Curso

Este curso tiene como objetivo proporcionar a los estudiantes los conocimientos y habilidades basicas respecto al uso de HTML, CSS y JavaScript para el desarrollo de sitios web. Ademas de comprender la arquitectura del desarrollo front-end.
